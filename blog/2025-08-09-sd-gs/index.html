SD-GS

论文阅读笔记：SD-GS。

[toc]

Pipeline

​ 用ScaffoldGS提供紧凑的静态几何结构，通过时间感知的变形场预测anchor的变形，最终生成当前时间下的GS进行渲染。

1. 摘要

现在方法：内存占用大，复杂运动描述能力弱。

对此提出：

  1. ==deformable anchor grid==: Scaffold GS的锚点策略；
  2. ==deformation-aware densification strategy==: 欠重建区域自适应补充锚点，静态区域去除冗余（How）；

讲述成果就围绕内存（尺寸减小60%）效率（FPS提升100%）质量（提升）。

2. 引言

3DGS显式几何表示 & CUDA加速渲染

现有方法：

  • 直接增加时间维度（4DGS）—— 无法利用cross-spatiotemporal correlations，所以内存消耗大
  • Deformable 3DGS（将每一帧的运动认为是基础3DGS的形变）—— 现有方法复杂运动处理能力较弱，渲染速度慢

解决方法：

​ 平衡效率和复杂运动的建模能力

  • 扩展 anchor-based scaffold representation from static to deformable 3DGS

    • SfM 初始化 deformable 3dgs
    • 时间感知架构

  • 静态GS基于固定的梯度阈值，应用到动态时，在静态区域容易产生大量冗余高斯，增加计算开销；在动态区域锚点不足，恢复不好。所以提出变形感知的致密方案，鼓励新的锚点在重建不足的高动态区域高效生长，同时减少静态区域的冗余。

3. 相关工作

4DGS，增加一个维度，需要大内存；

DeformableGS，变形场（时空相关性）。

GS Densification，根据梯度选择GS并clone；现有方法用大计算消耗换Densification；需要进一步探索densification方法。

4. 方法

模型结构

（1）Canonical Gaussian Model ℳ

使用 Scaffold-GS 表达静态结构，多个 anchor 构成。

每个 anchor 包含：x ∈ ℝ3，上下文特征 f ∈ ℝ32，旋转 q ∈ ℝ4（四元数），缩放因子 l ∈ ℝ6，偏移集合 O ∈ ℝk × 3：用于生成 k 个神经高斯。

每个 anchor 对应一组神经高斯，高斯位置计算：

{μ0, ..., μk − 1} = x + {O0, ..., Ok − 1} ⋅ l1 : 3

这些 anchor 初始化自 COLMAP 的 SfM 点，并在静态训练阶段优化。

（2）Anchor Deformation Field ℱ

建模动态变化，预测给定 t 下的 anchor 变形。

  • 输入：Canonical anchor 集合 ℳ ，时间 t ；
  • 编码器 ℋ：HexPlane 分解，多尺度特征编码；
  • 解码器 𝒟：三个 MLP 解码 anchor 变形。

最终得到：ℳ′ = ℳ + Δℳ。

变形后的 anchor 再用于生成时间感知的神经高斯。

（3）Neural Gaussians Generation Network 𝒩

从变形后的 anchor 中生成每一帧的神经高斯属性。

  • 输入：anchor 特征 f，相机方向 d⃗c，时间编码 ϕ(t) ；

  • 输出：除GS位置外的其他关键属性。 $$ \begin{cases} \alpha_i = F_\alpha(f, \vec{d}_c, \phi(t)) \\ c_i = F_c(f, \vec{d}_c, \phi(t)) \\ r_i = F_r(f, \vec{d}_c, \phi(t)) \\ s_i = F_s(f, \vec{d}_c, \phi(t)) \end{cases} \tag{2} $$

变形感知稠密化

传统方法每N次迭代计算 anchor 对图像 loss 梯度，大梯度区域效果差，添加新 anchor。但是短时间大变化区域平均下来的梯度会很小，那就无法有效加点。静态区域长期训练下反而累计比较大的梯度，造成静态冗余。

通过 anchor 的变形程度来控制 densification 权重，定义加权梯度为：

$$ \nabla g = \frac{\sum_{i=1}^N w_i \|\nabla_i^{2D}\|}{\sum_{i=1}^N w_i} \tag{3} $$

其中 $w_i = \alpha \frac{\|\Delta x_i\|}{s_x} + \beta \frac{\|\Delta l_i\|}{s_l} + \gamma \frac{\|\Delta q_i\|}{s_q}$。旋转差值使用四元数夹角计算：∥Δqi∥ = 2 ⋅ arccos (clip(|⟨qorig,qdef⟩|,0.0,1.0))

这样就可以让大变形高动态区域更容易被 densify，静态区域不浪费资源。

5. 结论

使用ScaffoldGS减少存储消耗并提高帧率。致密化提升了动态场景中锚点分布。

补充

Scaffold GS

anchor points（锚点）。对SfM-points进行体素化并得到初始锚点。

每一个锚点都有特征（f）、位置（x）、缩放因子（l）以及偏移量（O）。对于给定位置 xc 处的相机，根据锚点特征计算体素内的GS方法如下: {ci, ri, si, αi}j = 0k = MLP(f,σc,d⃗c)

  • σc = ∥x − xc∥2：锚点到相机位置的距离

  • $\vec{d}_c = \frac{\mathbf{x} - \mathbf{x}_c}{\| \mathbf{x} - \mathbf{x}_c \|_2}$：锚点指向相机的单位方向向量

并可以得到各个GS的位置，x 是锚点位置： {μ0, …, μk − 1} = x + {O0, …, Ok − 1} ⋅ l1 : 3 通过==从锚点特征预测GS属性、只保存锚点属性==，Scaffold GS极大地消除了GS冗余，减少了内存占用。

Deformable GS

spatial-temporal structure encoder （时空结构编码器，ℋ）

  • Hexplane 分解。将 (x,y,z,t) 投影到六个平面上 (x,y), (x,z), (y,z), (x,t), (y,t), (z,t)
  • 每个平面存储一个 2D-voxel 特征图，通过双线性插值采样得到当前点在当前平面对应的特征向量 fhi 。六个平面的特征图融合就得到 fh 。经过一个小MLP对 fh 进行融合与降维得到编码向量 fd 。

multi-head gaussian deformation decoder（多头高斯变形解码器，𝒟）

  • (Δ𝒳,Δr,Δs) = (ϕx(fd),ϕr(fd),ϕs(fd))

  • 用三个独立的MLP分别预测位置、旋转、缩放的变化。

  • 变形后的GS：G′ = {𝒳 + Δ𝒳, r + Δr, s + Δs, σ, C}

  • ==Hexplane 这样的特征编码能够让网络对时空相关性的理解更上一层楼。==

给定GS（G）以及时间戳（t），变形网络（ℱ）的功能：预测变形 ΔG = ℱ(G,t)。

为什么每个平面上是一个规则网格结构？

无法对连续的 2D 空间做每个点都存特征，所以必须离散化。

把空间中的特征提前编码在一个可学习的网格中——规则网格。

支持简单快速的双线性插值；GPU高速访问；对 2D 平面无偏覆盖，适合初始阶段。
